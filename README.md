# AI Debate Duel – GenLayer Mini-Game

AI Debate Duel is a multiplayer mini-game built for GenLayer’s community gatherings.  
Players compete by submitting arguments on a given topic, while the community determines the winner through simulated consensus.


## Game Overview

- Multiplayer debate game (3–10 players per room)
- Each player submits a short argument on a random topic
- Players vote on the best argument
- Winner is determined through simulated **Optimistic Democracy consensus**
- XP is distributed using a simulated **Intelligent Contract system**
- Leaderboard displayed after each round

## Features

- Random debate prompts (replayable weekly)
- Multiplayer room simulation
- Voting-based consensus mechanism
- XP leaderboard system
- Designed for 5–15 minute sessions


## GenLayer Concepts Implemented

### Intelligent Contracts
- Automatically handles XP distribution
- Records player actions and outcomes

### Optimistic Democracy
- Players vote to determine the winner
- Consensus decides final results
